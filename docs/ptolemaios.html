Basic Info.

  • Name: 프톨레마이오스의 세계지도 Ptolemy’s World Map

  • Who: 클라우디오스 프톨레마이오스 Claudius Ptolemy

    • Ptolemaios
    • Claudius Ptolemaeus

  • When: 150 CE

    • The original have been lost.

    • Many editions were made between 12th to 15th.

  • Where

    • Many libraries host various versions of manuscripts.

  • Size: 57 x 83 cm (Brotton 2014, 24)

    • Vatican Library, Rome, Italy, 13th
