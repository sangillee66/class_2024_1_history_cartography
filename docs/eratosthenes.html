Modified

March 30, 2024

Basic Info.

  • Name: 에라토스테네스의 세계지도 Eratosthenes’ World Map

  • Who: 키레네의 에라토스테네스 Eratostenes of Cyrene (276 - 194 BCE)

  • When: 194 BCE

    • The original has been lost.

    • Reproduced by Edward Bunbury (1811-1895) (Bunbury, 1879)

  • Where: No place

  • Size: Unknown
